# Data Warehouse & Data Analytics Project

Welcome to my ** Data Warehouse & Data Analytics Project ** repository
This project demonstrates a complete data warehousing and analytics solution right from building a data warehouse, derive actionable insights using the data analytics.
It includes all the best practices and processes that a standard industry follows.


## Project Requirements

### Objective
Develop a modern data warehouse using SQL Server to consolidate sales data, enabling analytical reporting and informed decision making.

#### Specifications
- **Data Sources**: Imported data from source systems ( erp and crm) in the form of .csv files.
- **Data Quality**: Cleanse and resolve data quality issues prior to analysis.
- **Integration**: Combine both sources into a single, user friendly data model designed for analytical queries.
- **Scope**: Focus on the latest dataset only; historization of data is not needed.
- **Documentation**: Provide clear documentation of the model to support both business stakeholders and analytics team.

  ### BI: Analytics & Reporting (Data Analytics)

  ####Objective

  Develop SQL- based analyticas to deliver detailed insights into:
  - **Customer Behaviour**
  - **Product Performance**
  - **Sales Trends**
 
  -  These insights empower stakeholders with key business metrics, enabling strategic decision-making.
